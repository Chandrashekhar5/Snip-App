# Snip-App

Snip-App is an iOS application designed to Authenticate the user and shows the list of songs. and is built with Swift/Objective-C and other technologies.

## Features
- Login Page
- Forgot Password Page
- Song's List page

## Tech Stack
- **Language**: Swift
- **Frameworks**: UIKit, XCTest etc.
- **Tools**: Xcode
